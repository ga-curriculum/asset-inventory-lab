## Scenario

Now that you've identified the issues in General Corporation's legacy asset inventory, it's time to start the reconciliation process. Your team has conducted a thorough audit of the company's current IT assets and compiled the data in a new spreadsheet, [General_Updated_Data.xlsx](../General_Updated_Data.xlsx).
Your task is to integrate this new data into the legacy inventory. You'll need to match the new records with the existing ones, update outdated information, fill in missing data, and resolve any conflicts or inconsistencies.

### Exercise steps

Open both the [General_Legacy_Inventory.xlsx](../General_Legacy_Inventory.xlsx) and [General_Updated_Data.xlsx](../General_Updated_Data.xlsx) files.
Start by matching the assets in the new data file with those in the legacy inventory. You can use unique identifiers like asset IDs or serial numbers to ensure you're updating the correct records.

For each matched asset:

- Update any outdated information in the legacy inventory with the new data. This might include things like location, assigned user, software versions, etc.

- Fill in any missing data points that are available in the new data file.

- If there are any conflicts between the legacy and new data (e.g., different purchase dates for the same asset), investigate and determine which information is correct. Document your reasoning in the "Inventory Issues" worksheet.

For any assets in the new data file that don't have a match in the legacy inventory, add them as new rows in the spreadsheet.

For any assets in the legacy inventory that don't have a corresponding entry in the new data file, try to determine their current status:

- If the asset has been decommissioned or is no longer in use, mark it as such in the inventory.
- If you're unsure about an asset's status, highlight it for follow-up investigation.

As you work through the reconciliation, keep track of all changes made in the "Inventory Issues" worksheet. Note the specific updates made to each asset record.

### Deliverable

An updated version of the [General_Legacy_Inventory.xlsx](../General_Legacy_Inventory.xlsx) file, with new data integrated and inconsistencies resolved.

An updated "Inventory Issues" worksheet documenting the changes made during reconciliation and any outstanding issues requiring further investigation.

Through this exercise, you'll gain practical experience in data reconciliation and updating complex datasets. You'll learn to identify matches between datasets, investigate and resolve data conflicts, and document your update process. These are critical skills for maintaining accurate and reliable asset inventories.
