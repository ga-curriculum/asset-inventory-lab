## Explore an existing inventory document

### Scenario

You've just joined the IT department of General Corporation, a mid-sized manufacturing company. One of your first tasks is to update the company's asset inventory, which hasn't been properly maintained for several years. The current inventory is stored in an old spreadsheet, and a quick glance reveals numerous inconsistencies, gaps, and outdated information.

Your manager has tasked you with reconciling this legacy inventory with fresh data collected by your team. You'll need to identify the issues in the current inventory, integrate the new data, and create an updated, accurate record of all the company's IT assets.

### Exercise steps

1. Open the provided [General Legacy Inventory](https://docs.google.com/spreadsheets/d/1erPo5Ngah_uw0-LIzw-UsunqEwzLTQLzDUWScFPp0e0/edit) file. This spreadsheet contains the company's current asset inventory. Make a copy of the file on your own Google Drive so that you can edit it. You can do this by clicking on **File** in the menu and selecting **Make a copy**. Name the file something like `General Legacy Inventory - Your Name`.

2. Ensure the file is publicly shared so your instructor can access it when you submit your work.

3. Familiarize yourself with the structure and content of the spreadsheet. Note the different columns (for example, **Asset ID**, **Type**, **Location**, **Assigned To**, **Purchase Date**, and so on). Also note the types of assets included (**hardware**, **software**, **licenses**).

4. Add a new sheet named **Inventory Issues** in the existing spreadsheet.

5. Start analyzing the data to identify potential issues. As you encounter issues, list all them in the **Inventory Issues** sheet. Document the specific cells or rows they pertain to. This will serve as a guide for the data reconciliation process.

   - Check for missing or incomplete data in each column. Are there any blank cells or placeholders like "N/A" or "TBD"?

   - Look for inconsistencies in data formatting. For example, are dates in different formats? Are asset names spelled differently?

   - Identify any outdated information, such as assets assigned to employees who no longer work at the company.

   - Note any assets that seem to be duplicated or have conflicting information.

   - You won't be able to resolve these issues yet, just document them for now!

## When you're done

When you're complete, you should have your own copy of the **General Legacy Inventory** Google Sheet that contains an **Inventory Issues** sheet. The **Inventory Issues** sheet should detail the problems you've identified in the legacy inventory data so far.

By the end of this exercise, you'll have a clear understanding of the state of the legacy inventory and the challenges you'll need to address in the reconciliation process. This foundational analysis will guide your work in the subsequent exercises, where you'll integrate new data and create an updated, accurate inventory.
