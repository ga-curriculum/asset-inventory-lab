## Lab requirements

The following requirements describe the necessary deliverables for the Asset Inventory Lab.

| Criteria                                                                                                                                                                                                          | Points |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| The **General Reconciled Inventory** file was shared publicly so that it is able to be viewed by the instructor on their first attempt when grading.                                                              | 5      |
| At least 80% of the entries in the final reconciled inventory sheet are accurate.                                                                                                                                 | 30     |
| At least 80% of the changes made in the final reconciled inventory sheet are documented in the **Inventory Issues** sheet.                                                                                        | 15     |
| The final reconciled inventory document includes **General Reconciled Inventory** in the title.                                                                                                                   | 5      |
| The **Inventory Update Report** was shared publicly so that it is able to be viewed by the instructor on their first attempt when grading.                                                                        | 5      |
| The **Inventory Update Report** contains an **Introduction** section that briefly explains the objectives and scope of the inventory update project.                                                              | 5      |
| The **Inventory Update Report** contains a **Methodology** section that describes the step you took to reconcile the legacy data, integrate new information, and incorporate intangible assets.                   | 5      |
| The **Inventory Update Report** contains a **Results** section that highlights the key improvements in the updated inventory compared to the legacy data.                                                         | 5      |
| The **Inventory Update Report** contains a **Recommendations** section that provides suggestions for maintaining the accuracy of the inventory from now on.                                                       | 5      |
| The **Inventory Update Report** contains a **Conclusion** section that summarizes the overall impact of the updated inventory on General Corporation's IT asset management capabilities and potential next steps. | 5      |
| The **Inventory Update Report** contains a link to the final version of your **General Reconciled Inventory** file.                                                                                               | 5      |
| The **Inventory Update Report** is formatted in a clear manner.                                                                                                                                                   | 5      |
| The **Inventory Update Report** uses clear and professional language.                                                                                                                                             | 5      |
| **Total possible points**                                                                                                                                                                                         | 100    |
